## 快速使用

YARN
```shell
npm i -g yrm
yarn global add @careteen/cli
```
THEN

![@careteen-cli-example-1](https://careteenl.github.io/images/careteen-cli-example-1-new-mini.gif)

### 全局安装

```shell
yarn global add @careteen/cli
```

### 查看所有命令帮助
```shell
careteen --help
```

### 下载项目
```shell
careteen create my-project
```
